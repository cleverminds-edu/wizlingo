================================================================================
                        DATABASE SCHEMA
================================================================================

Model: Certificate (existing in prisma/schema.prisma)

fields:
  - id: String @id @default(cuid())
  - studentId: String
  - badgeType: BadgeType
  - verifyCode: String @unique @default(cuid())
  - issuedAt: DateTime @default(now())
  - student: Student @relation(...)

constraints:
  - @@unique([studentId, badgeType]) - One certificate per badge per student

Relationship:
  - Belongs to Student model
  - Auto-created when Badge is awarded
  - Links via studentId and badgeType

================================================================================
                           HOW TO USE
================================================================================

1. GENERATE CERTIFICATE
   POST /api/certificates/generate
   Body: { studentId: string, badgeType: string }

2. DOWNLOAD CERTIFICATE
   GET /api/certificates/download?studentId=...&badgeType=...

3. VERIFY CERTIFICATE
   GET /api/certificates/verify?code=ABC123DEF456

4. VIEW CERTIFICATE
   Navigate to /certificate/[verifyCode]

5. CUSTOMIZE
   Edit templates in lib/certificate-templates.ts
   Edit styles in lib/pdf-certificate-generator.ts
