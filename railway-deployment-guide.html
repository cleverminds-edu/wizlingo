❓ Troubleshooting

❌ "Build Failed"

Check the logs:
  1. Click "Deployments" tab
  2. Look for red "Failed" status
  3. Click it to see error message
  4. Common fix: Missing environment variable

❌ "Database Connection Error"

Verify DATABASE_URL:
  1. Check "Variables" tab in app service
  2. DATABASE_URL should exist (Railway creates it)
  3. If missing, delete PostgreSQL service and add it again

❌ "Can't Access App"

Wait and refresh:
  1. First deployment takes 10-15 minutes
  2. Wait for green "Live" status in Deployments
  3. Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

❌ "500 Error on Pages"

Check app logs:
  1. In app service, click "Logs" tab
  2. Look for red error messages
  3. Most common: Missing env variable

🔗 Optional: Custom Domain Setup

Want wizlingo.edvanta.co.in instead of the long Railway URL?

Do this AFTER verification (not required to launch):

  1. In app service, go "Settings"
  2. Find "Domains" section
  3. Click "Add Custom Domain"
  4. Enter: wizlingo.edvanta.co.in
  5. Railway shows you a CNAME record to add
  6. Go to your DNS provider (wherever you manage edvanta.co.in)
  7. Add the CNAME record Railway provided
  8. Wait 5-15 minutes for DNS to propagate

⚠️ Not urgent for beta launch

You can skip this initially and use the Railway URL. Set up custom domain later when schools are onboarded.
